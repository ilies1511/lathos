# Project Setup

## 1. Build the Project

```bash
make
```

## 2. Start the Development Server

```bash
npm run dev
```

## 3. Load the Extension in Chrome

1. Open **Chrome** and go to `chrome://extensions/`
2. Enable **Developer Mode** (toggle in the top right)
3. Click **Load unpacked**
4. Select the following folder:

```
lathos/core/dist/
```
